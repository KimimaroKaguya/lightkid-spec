# LightKid UI Design Specification

## Overview
LightKid features a modern, clean, and playful interface designed for parents. The design prioritizes ease of use (one-handed operation), clear data visualization, and a comforting aesthetic using soft pastel colors.

## Design System

### 1. Color Palette
Soft, calming colors to reflect the nature of childcare.

- **Primary Brand**: `Soft Blue` (#64B5F6) - Used for primary actions, active states.
- **Secondary**: `Mint Green` (#81C784) - Used for success states, health indicators.
- **Accent**: `Warm Yellow` (#FFD54F) - Used for milestones, highlights.
- **Background**: `Off-White` (#F5F5F7) - Reduces eye strain compared to pure white.
- **Text**: `Dark Grey` (#37474F) - Softer than pure black.

### 2. Typography
Clean sans-serif fonts for readability.

- **Family**: System Default (San Francisco on iOS, Roboto on Android) or **Inter**.
- **Headers**: Bold, rounded soft edges if possible.
- **Body**: Regular weight, 16px base size for legibility.

### 3. Layout Principles
- **Mobile-First**: Designed for vertical phone screens.
- **Bottom Navigation**: Primary navigation (Home, Growth, Milestones, Settings) at the bottom for thumb reachability.
- **Card-Based Content**: Information grouped into distinct white cards with soft shadows `box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1)`.
- **Whitespace**: Generous padding (16px/24px) to avoid clutter.
